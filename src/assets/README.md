# Assets Folder

Esta carpeta contiene los recursos visuales del proyecto.

## Logo
- Coloca tu archivo de logo aquí con el nombre `logo.png`
- El logo se mostrará en el header con un tamaño de 48x48px
- Formato recomendado: PNG con fondo transparente

## Imágenes del Hero
Las imágenes del carrusel se cargan desde URLs externas configuradas en `HeroCarousel.jsx`
